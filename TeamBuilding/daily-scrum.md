# daily scrum
- 今のところの自分が良いと考えるデイリースクラムの概要
```
▼目的
・スプリントゴール達成のための相互の状況把握 (タスク進捗、コンディション等)

▼時間枠
・15min デイリースクラム
・15min 二次会 (相互の状況把握した結果SBIの見直しや詳細議論が必要な場合のみ)

▼進め方
・Good & New！

・デイリーの目的、スプリントゴールのおさらい

・一人ずつ以下を共有
　・昨日やったこと
　・今日やること
  ・スプリントゴール達成の阻害になっていること
　・その他共有したいこと

・最後に話し忘れたことがあるか確認

▼参考
[スクラム開発におけるデイリースクラムの基本ルール・失敗例](https://do-scrum.com/daily/)
```
