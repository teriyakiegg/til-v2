# double diamond
```
ダブルダイヤモンドというフレームワークをご存知でしょうか？
ここでは詳しく説明しませんが、簡単にいうと「発散と収束を繰り返し、課題の特定と解決の精度を高めるためのデザインプロセス」です。
前半の発散（探索）と収束（定義）で課題を特定し、後半の発散（展開）と収束（提供）で解決方法を見つけます。
ref. https://uxdaystokyo.com/articles/glossary/doublediamond/
```
