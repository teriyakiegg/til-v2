vscodeでGoのファイル開いた時、vscodeが促してきたインストールとかを言われるがままに入れていたら、  
気付いたらGoのLanguage Server(gopls)を使う設定になってた。  
Settingsでgolangで検索したら↓にチェック入ってた
```
Go: Use Language Server
Use the Go language server "gopls" from Google for powering language features like code navigation, completion, formatting & diagnostics.
```
