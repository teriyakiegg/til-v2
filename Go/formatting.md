# formatting
go fmtするとgo標準のフォーマットをしてくれるらしい。これで書き方宗教戦争から開放される  

## gofmtとgoimportsの違い
```
goimportsは、公式で提供しているgofmtの上位互換ツールです。
コードのフォーマットに加えて、importの自動解決（パッケージ名のアルファベット順にソート、不要なパッケージの削除など）の機能を備えています。
```
